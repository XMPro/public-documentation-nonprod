Container Security Scan - XMPro 4.6.1

Registry: xmpro.azurecr.io Version: 4.6.1 Release Date: 13 May 2026

Executive Summary

All Critical and High severity findings across XMPro 4.6.1 container images have been investigated and confirmed as false positives (see Known False Positives). No actionable Critical or High vulnerabilities exist in any XMPro container image.

Eighteen new false positives were assessed since the 4.6.0 release; the remaining entries carry forward from earlier releases.

Status Count
Total documented false positives at 4.6.1 54
New since 4.6.0 18
Carried forward from 4.6.0 36
Note

A CVE listed on Known False Positives means a security scanner has flagged the vulnerability in an XMPro container image, but engineering has reviewed the issue and confirmed it does not affect XMPro at runtime. Common reasons include: the vulnerable code path is never executed, the affected library is transitively installed but not invoked, or the vulnerability requires preconditions that do not exist in XMPro deployments. Genuine vulnerabilities (true positives) are addressed in the release itself and do not appear on this page.
