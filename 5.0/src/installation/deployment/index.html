2. Install XMPro

XMPro Platform can be installed using a range of Infrastructure, please choose the option below that best suits your requirements:

Cloud Deployment

Azure Terraform Deployment

Recommended for XMPro v4.5+

Modern Infrastructure as Code deployment using Terraform. This approach provides:

  • Reproducible infrastructure deployments
  • Version-controlled infrastructure changes
  • Production-ready defaults and best practices
  • Support for existing database scenarios
  • Comprehensive monitoring and logging

Azure ARM Template Deployment (Legacy)

For XMPro v4.4 and earlier versions

Traditional ARM template deployment approach. This method is maintained for:

  • Existing v4.4 installations requiring maintenance
  • Legacy upgrade scenarios
  • Organizations with ARM template requirements

Note: New deployments should use the Terraform approach above for the latest features and improved deployment experience.

Other Deployment Options

AWS Deployment

Deploy XMPro on Amazon Web Services using Kubernetes and Helm charts.

Windows Server 2022 Deployment

On-premises deployment for Windows Server environments.
