Change TypeDescription
Feature

Classify your Recommendation Alerts to better group for action or analyze for trends.

  • The classification options are Asset, Process, KPI, Work Order, or Work Request.
  • Set the classification using new versions of the existing Run and Update Recommendation Agents.
  • Use the extended classification identifiers as filters in these enhanced Blocks: Alert Analytics, Alert Discussion, and Alert List.
Feature

The last of the new Blocks to make your interactions with Recommendation Alerts composable:

  • Alert Action - add buttons to take actions on an alert, such as Share
  • Alert Survey - add buttons for feedback from users on a Recommendation
