App Designer

Change Type Description
Fix In v4.4.7, I can't configure a data source to use my stored procedure due to an error that the primary key is required.
Data sources for stored procedures without a select statement will not require a primary key.
Fix In v4.4.17, when I change the date range on my timeseries analysis, the markers I placed lose their data value label.
Markers stay visible, along with their value.
Fix In v4.4.18, when I view a Data Grid at runtime, the datetimes stored as UTC are not converted to my local time zone.
The Data Grid converts UTC datetime values to user's local time zone.
Fix When a new level of business role was added, I can no longer see recommendation alerts for my "grandparent" business role.
A user belonging to any child business role will have run access to recommendation alerts of the parent business role.
Fix In v4.4.16, the Chart Block's Orientation drop-down is empty.

The Chart's Orientation drop-down contains options for Vertical and Horizontal.
Fix In v4.4.16, when defining the Color for a chart series, I no longer have the option to choose a value from the data source.
For a Chart Block with a data source bound, the Series Setting's Color dropdown allows selecting of variables, parameter or columns from the bound data source.

Stream Host

Change Type Title
Fix In v4.4.20, my Stream Host shows a fatal error when I accidentally used the wrong username in my SQL User variable and published a data stream that included an agent that uses the variable.
The stream host will not be affected by an incorrect variable value.
